# 🚀 MongoDB Basics + Mini WhatsApp Project

This repository contains my complete learning journey of MongoDB, Mongoose, and Express.js along with a mini project.

---

## 📱 Mini WhatsApp Project (Express + MongoDB)

A CRUD-based web application where users can create, edit, view, and delete chats.

### ✨ Features
- ✉️ Create new chats  
- ✏️ Edit existing chats  
- ❌ Delete chats  
- 📄 View all chats  

### 🛠 Tech Stack
- Node.js  
- Express.js  
- MongoDB  
- Mongoose  
- EJS  

---

## 📚 Topics Covered

- MongoDB Basics  
- Collections & Documents  
- CRUD Operations (Create, Read, Update, Delete)  
- Query Operators ($gt, $lt, $in, $or, etc.)  
- Nested Documents (Nesting)  
- Mongoose (Schema, Model)  
- CRUD using Mongoose  
- Async/Await & Promises in Node.js  
- Custom Error Handling
